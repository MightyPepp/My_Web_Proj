# My_Web_Proj
My project to learn web dev and data structures and algorythms

В main.go только запуск сервера или чего-то такого.

В server/ лежит логика бэкенда, маршрутизация в итоге будет через mux, пока ручками перебираем строку запроса.

В static/ просто для тестирования три сраницы-пустышки.

В stack/ рабочий учебный самопиный стек, работа которого в дальнейшем должна стать примером для изучения работы бека и отправления данных на фронт.
